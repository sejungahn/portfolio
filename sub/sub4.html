Overview

총 3개의 사이트를 작업했어요. 기존에 계속 유지보수하던 사이트의 리뉴얼은 어렵지 않게 진행되었습니다.

대표 국문 메인 페이지에서는 기존에 사용하던 owl.js를 swiper.js로 교체하는 작업을 진행했어요. 퍼블리싱 작업 시 웹 접근성과 웹 표준을 준수하여 꼼꼼하게 작업했습니다. 대체텍스트를 부여하는 등 키보드 접근성에도 신경을 써서 진행했습니다.

서브페이지에서는 레이아웃 변경에 따라 디자인에 맞춰 새로운 레이아웃으로 작업하였고, 컨텐츠는 신규로 생성되는 10페이지 정도의 페이지를 제외하고는 내용이 동일한 컨텐츠를 재활용하여 CSS만 변경하는 정도로 작업했습니다. 영문 사이트는 국문과 동일한 디자인으로 수월하게 마무리하였습니다.

출입기자단 사이트는 서브페이지가 없는 심플한 단일 사이트로 어려움 없이 마무리했습니다. 작업 시 3명의 퍼블리셔와 함께 진행했는데, 팀원들 중 팀장으로 업무를 균일하게 배분하여 작업을 진행했습니다. 진행한 작업물들을 보며 피드백을 주고, 기한 안에 작업을 마무리할 수 있게 조율하며 협업했어요.
