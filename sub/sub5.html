Overview

2개의 템플릿 작업과 신규 템플릿을 포함한 총 12개의 템플릿을 활용하여 98개의 사이트를 구축했어요. 2차로 진행된 프로젝트라 기존 10개의 템플릿을 제외하고 추가로 2개의 템플릿을 새롭게 작업했어요. 템플릿1과 템플릿2 두 사이트 모두 원활하게 마무리하였습니다.

템플릿1은 템플릿2 보다 기능적인 부분이 더 많이 포함된 사이트여서 작업에 좀 더 시간이 소요되었어요. 메인 페이지에서는 팝업, 스크롤, 메뉴 같은 간단한 스크립트로 작업하였고, 다른 곳에서도 사용할 수 있게 모듈화했고, 모든 슬라이드는 swiper.js를 활용하여 작업했어요. 공지사항, 학사일정 등 서버 연동되는 컨텐츠들은 HTML로 선 작업 후 JSTL 재코딩하여 마무리했습니다.

서브페이지는 PSD를 참고하여 레이아웃과 예시 컨텐츠를 적용했어요. 템플릿1 에서 사용하는 신청용 게시판 write 페이지나 설문조사 폼을 함께 구현했습니다.

총 98개의 사이트는 팀원들과 함께 기존의 템플릿을 활용하여 리뉴얼 작업을 진행했어요.
프로젝트 초기에는 분배에 어려움이 있었지만, 팀원들과의 소통을 통해 적절하게 작업을 분배하고, 기한보다 더 빠르게 프로젝트를 성공적으로 완료할 수 있었어요.
